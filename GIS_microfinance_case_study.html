# PROBLEM STATEMENT

# Kiva.org is an online crowdfunding platform extending financial services to poor people around the world. 
# Kiva lenders have provided over $1 billion dollars in loans to over 2 million people.
# Kaggle Dataset as on 31 December 2021.

# Kiva reaches some of the most remote places in the world through their global network of "Field Partners". 
# These partners are local organizations working in communities to vet borrowers, provide services, and administer loans.
# In this programing snippet, and deriving inspiration from the kaggle dataset on kiva, I demonstrate the following:

# 1. Use the open source kiva dataset from kaggle, import the .shp files, and display the loans distribution in the world.

# 2. I investigate the Kiva loans in India to identify regions that might be outside of Kiva's current network, 
#    in order to identify opportunities for recruiting new Field Partners. For this purpose I overlay India.kml file onto
#    the existing .shp file, which demonstrates the flexibility offered by python to display various file formats on a single
#    screen.
